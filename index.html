What’s in the Image? A Deep-Dive into the Vision of Vision Language Models

Omri Kaduri*, Shai Bagon*, Tali Dekel
Weizmann Institute of Science
*Indicates Equal Contribution

How Visual-Language Models see?

In this paper, we focused on understanding how Vision-Language Models (VLMs) process visual information. We focus on the attention modules across layers, and reveal that:

  1. Query tokens are used by the model to store global image information.
  2. Cross-modal information flow is predominantly influenced by the middle layers.
  3. Fine-grained details are directly extracted from image tokens in a spatially localized manner.
In this project page, we provide a brief overview of our findings and demonstrate the potential of our observations in facilitating efficient visual processing in state-of-the-art VLMs.

Visual Information Knockout

We aim to understand how Vision-Language Models process visual information. We reveal that these models are able to generate descriptive responses while only attending directly to query tokens (e.g., "describe the image"), without direct access to image tokens.
Moreover, the query tokens must pull global image information, otherwise the model fails to describe the image.
Next, we describe the llm-as-a-judge evaluation protocol, which allows us to evaluate the model's ability to identify objects in the image.

LLM-as-a-judge for Knockout Evaluation

Automatically counting the number of identified or hallucinated objects in free-text paragraphs is challenging.
Therefore, we propose the LLM-as-a-judge evaluation protocol, which allows us to automatically assesses object identification and hallucinated objects in generated responses under different knockout configurations.

Visual Information Knockout Evaluation

Analyzing visual information flow via attention knockout:
(a) The VLM employs causal masking allowing generated and query tokens to gather information from image tokens, but not vice versa.
We analyze three knockout configurations: (b) Image-to-generated: visual information flows to generated tokens only through query tokens, (c) Image-to-query: blocks query tokens from accessing image information, and (d) Image-to-others: blocks image tokens from all other tokens. (e) Evaluation of model responses under each knockout configuration reveals that Image-to-generated knockout achieves a 0.4 F1 score despite indirect image access, while Image-to-query fails completely, demonstrating query tokens' essential role as global image descriptors.
(f) We expand previous experiments by knocking out attention, starting from layer l. Results highlight a consistent drastic rise in F1 scores in the mid-layers, suggesting their critical role in visual information processing.
